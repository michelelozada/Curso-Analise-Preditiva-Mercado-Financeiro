## <img src="https://www.michelelozada.com.br/images_github/analise_preditiva/square.jpg" style="width:15px; vertical-align: middle; margin-right: 10px;"> Descrição  
Este repositório centraliza atividades práticas e notas de estudo implementadas durante minha participação no curso de Análise Preditiva Aplicada ao Mercado Financeiro com Python (2ª edição), que foi promovido em outubro de 2025 pelo grupo de pesquisa [Matemática Aplicada à Visão Computacional (Mavicom/IFS)](https://www.instagram.com/mavicom.ifs/) com apoio da Pró-Reitoria de Extensão do [Instituto Federal de Sergipe](https://www.ifs.edu.br/).  

O curso teve por objetivo compartilhar conhecimentos sobre conceitos fundamentais e ferramentas de análise quantitativa aplicadas a dados financeiros, através de simulações didáticas.
